Detailed Description

Game::Game ( )

Constructs the game, loads fonts, initializes UI and pieces.

Constructs the Game object, loads fonts, initializes UI text, sets up menu buttons, creates starting pieces, and prepares the game window.

This constructor:

  • Loads the game font
  • Sets up title, subtitle, credits text
  • Initializes player pieces and textures
  • Creates all main menu buttons
  • Sets initial game state to MainMenu

Behavior depends on the current game state:

  • Placement: pick up unplaced pieces
